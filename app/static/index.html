When to use each level

  • Strict (0.95) — Production services, auth, payments, user data, anything shipped to customers. We only interrupt when we’re highly confident there’s a real vulnerability.
  • Balanced (0.90) — Everyday internal tools, scripts, and learning. Good default for most chats.
  • Quiet (0.85) — POCs, spikes, demos, and brainstorming. Fewer regen prompts; review code yourself before reuse.
